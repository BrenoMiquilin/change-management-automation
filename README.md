# 🔄 Change Management Automation

## 📌 Overview
Automation designed to streamline change management workflows by sending dynamic notifications based on predefined conditions.

## ⚙️ How it works
- Triggered when a task is completed in Microsoft Planner
- Checks specific conditions (bucket, user, flags)
- Sends emails to responsible teams
- Sends notifications via Microsoft Teams
- Dynamically adjusts recipients based on selected flags

## 🛠️ Tools
- Power Automate
- Microsoft Planner
- Outlook
- Microsoft Teams

## 🎯 Results
- Eliminated manual communication steps
- Increased reliability of change processes
- Improved coordination between teams
- Reduced human error
